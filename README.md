# pawn-ls
This project is the implementation of language server for PAWN Language.

## Features
- [ ] Code Completion
- [ ] Hover
- [ ] Signature
- [ ] Diagnostic

## Requirements
- CMake version 3.11 or higher
- C compiler
- C++ compiler that supports C++17 standards or higher
